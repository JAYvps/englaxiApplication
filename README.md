这是一款设计精良、体验流畅的沉浸式英语学习游戏。我们坚信，它独特的学习模式为市场注入了新的活力，并展现了巨大的发展潜力。

我们后续也会基于目前的故事继续开发 **我们相信，所以我们还在继续；**

## 🚀 运行指南

### 环境要求

*   **Node.js**: 推荐使用 LTS 版本的 `18.x`、`20.x` 或更高版本。

### 启动步骤

_（说明：尽管项目文件可能已包含依赖项，我们仍建议您遵循标准安装流程以确保环境一致性。）_

**1. 启动后端服务**

```sh
# 进入后端目录
cd backend

# 安装依赖
npm install

# 启动服务 (服务将运行在 http://localhost:3001)
npm run dev
```

**2. 启动前端应用**

```sh
# 在项目根目录 (englishApp) 打开一个新的终端

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 🚀技术架构
**前端：**
  * 核心框架: React
  * 构建工具: Vite
  * 编程语言: TypeScript
  * 样式: Tailwind CSS
  * 动画库: Framer Motion
  * 图标: Lucide React
  * AI 功能: Google GenAI SDK (是的 我们已经在Demo中引入 但因为特殊原因我们并没有使用它)

**后端：**
   * 核心框架: Express.js
   * 数据库 ORM(对象关系映射): Prisma
   * 数据库: SQLite (在开发环境中使用)
   * 编程语言: TypeScript

这是一个基于 React + Node.js 的现代化全栈 TypeScript 项目，前端注重用户体验和视觉效果，后端则使用 Express 和 Prisma提供了稳定高效的数据服务

## 🌟 2025年12月24日 更新日志 (v1.3)

我们对应用进行了大规模的更新与重构，旨在提供更丰富、更连贯的学习体验。

### ✨ 新功能 (New Features)

*   **完整的用户系统**: 新增了用户 **登录、注册与密码找回** 功能 (`AuthScreen.tsx`)，为实现个性化学习路径和数据同步奠定了基础。
*   **AI 助记功能正式上线**: 我们现在真正地集成了 Gemini AI！新增 **“词源助记”** 功能，利用 AI 帮助用户更科学、高效地记忆单词。
    *   **如何启用**: 只需在项目根目录创建 `.env.local` 文件，并填入您的 `VITE_GEMINI_API_KEY="YOUR_API_KEY"` 即可体验。
*   **“勋章”道具系统**: 引入了全新的 **“勋章”** 道具。玩家可以通过完成挑战获得，并用它来解锁后续的故事章节和关卡，极大地增强了游戏的可玩性。

### 🔄 核心重构与优化 (Core Refactors & Optimizations)

*   **学习路径重构**: 将原有的“地图选择”模式全面升级为更具沉浸感的 **“章节故事线”** 模式。用户将跟随主线故事，一步步解锁并挑战精心设计的关卡。
*   **后端字段与 API 升级**: 为了支持“词源助记”功能，我们已在后端数据库中添加相应字段，并更新了 API 接口。
*   **整体交互与视觉优化**: 对应用的 **CSS 样式和动画效果** 进行了全面微调，提升了界面的美观度和操作的流畅性。
