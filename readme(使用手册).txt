先决条件：NodeJS 推荐使用 18.x 或 20.x（或更高）的 LTS (长期支持)版本。
   1. 启动后端服务：
       +文件夹下backend 目录：安装后端依赖 npm install
       +启动后端服务：npm run dev (这会启动一个在 http://localhost:3001 上运行的服务)

   2. 启动前端应用：
       +打开另一个新的终端或命令行窗口。
       +确保您在项目的根目录 (englishApp下)。
       +安装前端依赖：npm install 
       +启动前端开发服务：npm run dev
